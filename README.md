# DisplaySwitcher 🖥

> 🇩🇪 **Deutsche Version weiter unten** | German version below ↓

---

Do you use multiple monitors and switch between different setups regularly? For example, three screens at your desk during the day, and just the TV in the evening for your home cinema? DisplaySwitcher saves those setups as profiles and lets you switch between them in two clicks – including automatic switching of your audio device.

---

## Download

➡️ **[Click here to download DisplaySwitcher](../../releases/latest)**

Simply download `DisplaySwitcher.exe` and run it. That's it.  
No installation, no setup, nothing else required.

Requirements: Windows 10 or Windows 11 (64-bit)

---

## What can DisplaySwitcher do?

- **Save monitor profiles** – set up once which monitors should be active in each scenario
- **Switch with a right-click** – right-click the icon in the taskbar, select a profile, done
- **As many profiles as you like** – e.g. "Office", "Home Cinema", "Presentation"
- **Resolution & arrangement** are saved per profile and automatically restored
- **Automatically switch audio output** – e.g. switch to speakers for the home cinema profile, headset for the office profile
- **Launch automatically with Windows** (optional)

## What can DisplaySwitcher **not** do?

- ❌ No keyboard shortcuts to switch profiles (planned for a future version)
- ❌ Windows only – no Mac, no Linux

---

## How to set it up

1. Start `DisplaySwitcher.exe` – a small icon appears in the bottom-right corner of the taskbar
2. **Right-click** the icon → "Configure profiles" → "Create new profile"
3. Give the profile a name (e.g. "Home Cinema" or "3 Screens")
4. Select which monitors should be active in this profile
5. Optionally: choose speakers and microphone for this profile
6. Click "Save" – done

Repeat this for every setup you use. After that, a single right-click on the icon is all it takes to switch.

Since there is currently no way to reposition monitors within DisplaySwitcher, it's best to arrange your screens in Windows first (Settings → System → Display) and then save that arrangement as a profile.

> **Which monitor is which?**  
> Press the Windows key → Settings → System → Display → click **Identify**.  
> Windows briefly shows a number on each screen – those same numbers appear in DisplaySwitcher.

---

### If a monitor is not detected

Right-click `DisplaySwitcher.exe` → "Run as administrator".  
If that helps and you don't want to do it every time:  
Right-click → Properties → Compatibility → check "Run this program as an administrator".

# DisplaySwitcher 🖥 – Deutsche Version

> 🇬🇧 **English version above** ↑

Du hast mehrere Monitore und wechselst regelmäßig zwischen verschiedenen Setups? Zum Beispiel tagsüber drei Bildschirme am Schreibtisch, abends nur der Fernseher fürs Heimkino? DisplaySwitcher speichert solche Setups als Profile und lässt dich mit zwei Klicks zwischen ihnen wechseln. Neben den Bildschirmen können auch Audiogeräte automatisch mitgewechselt werden.

---

## Download

➡️ **[Hier klicken um DisplaySwitcher herunterzuladen](../../releases/latest)**

Einfach die `DisplaySwitcher.exe` herunterladen und starten. Fertig.  
Keine Installation, kein Einrichten, nichts weiter nötig.

Voraussetzung: Windows 10 oder Windows 11 (64-bit)

---

## Was kann DisplaySwitcher?

- **Monitor-Profile speichern** – du legst einmalig fest, welche Monitore in welchem Setup aktiv sein sollen
- **Per Rechtsklick wechseln** – Rechtsklick auf das Symbol in der Taskleiste, Profil auswählen, fertig
- **Beliebig viele Profile** – z. B. „Büro", „Heimkino", „Präsentation"
- **Auflösung & Anordnung** werden je Profil gespeichert und automatisch wiederhergestellt
- **Audiogerät automatisch mitschalten** – z. B. beim Heimkino-Profil auf Lautsprecher wechseln, beim Büro-Profil auf Headset
- **Automatisch mit Windows starten** (optional)

## Was kann DisplaySwitcher **nicht**?

- ❌ Kein Wechsel per Tastenkürzel (soll zukünftig hinzugefügt werden)
- ❌ Nur für Windows – kein Mac, kein Linux

---

## So richtest du es ein

1. `DisplaySwitcher.exe` starten – unten rechts in der Taskleiste erscheint ein kleines Symbol
2. **Rechtsklick** auf das Symbol → „Profile konfigurieren" → „Neues Profil erstellen"
3. Dem Profil einen Namen geben (z. B. „Heimkino" oder „3 Bildschirme")
4. Auswählen, welche Monitore in diesem Profil aktiv sein sollen
5. Optional: Lautsprecher und Mikrofon für dieses Profil auswählen
6. Auf „Speichern" klicken – fertig

Wiederhole das für jedes Setup das du nutzt. Danach reicht ein Rechtsklick auf das Symbol, um zu wechseln.
Da aktuell noch keine Möglichkeit besteht, Bildschirme in DisplaySwitcher zu verschieben, bietet es sich an, die Bildschirme über Windows so anzuordnen wie man es braucht (Einstellungen → System → Anzeige) und im Anschluss als Profil zu speichern.

> **Welcher Monitor ist welcher?**  
> Windows-Taste → Einstellungen → System → Anzeige → **Identifizieren** drücken.  
> Windows zeigt kurz eine Zahl auf jedem Bildschirm an – dieselben Nummern siehst du auch in DisplaySwitcher.

---

### Falls ein Monitor nicht erkannt wird

Rechtsklick auf `DisplaySwitcher.exe` → „Als Administrator ausführen".  
Wenn das hilft und du das nicht jedes Mal machen willst:  
Rechtsklick → Eigenschaften → Kompatibilität → Häkchen bei „Programm als Administrator ausführen".
